Package: org.simplemes.eframe.test.page

[Groovy] Class AbstractPage

      • org.simplemes.eframe.test.page.AbstractPage 

      • class AbstractPage
extends geb.Page

        The general base class for all pages tested in the framework. Provides basic methods to wait for Ajax completion on load and when needed. If the variable 'waitForAjaxOnLoad=true', then the page will wait until the jQuery Ajax methods are done. On your constructor, you would set this using the code: 

         public LoginPage() {
   // No need to wait for Ajax finish on load.
   waitForAjaxOnLoad=false
 }

        This page defines these content sections:

        • messages - The error/info/warning message section. See MessagesModule for details.
        • button - A parameterized generic toolkit button (e.g. button('BUTTON').click()). Supports clicked() and text() methods.
        • toggleConfigButtons - The main button to toggle configuration buttons on a page.
        • logoutButton - The logout button.
        • dialog0 - The first dialog.
