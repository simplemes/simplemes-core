[Groovy] Class StandardModelAndView

      • org.simplemes.eframe.controller.StandardModelAndView 

      • @groovy.transform.ToString(includePackage: false, includeNames: true, includeSuper: true)
class StandardModelAndView
extends io.micronaut.views.ModelAndView

        Defines an enhanced version of the model and view object used to display a page from a controller method. Adds user principal details and marker data to the normal ModelAndView.
