Package org.simplemes.eframe.exception

Package org.simplemes.eframe.exception Description

Exception and support classes.
  • Class Summary 
    Class Description
    MessageHolder This class encapsulates multiple response messages (error, etc) and allows controllers to render them in a consistent format.
    MessageHolderSerializer This is the Jackson serializer used for a MessageHolder.
  • Exception Summary 
    Exception Description
    BusinessException Generic business exception that is based on error numbers.
    MessageBasedException A general-purpose message exception that uses error numbers and messages from the .properties file to localize the message.
    SimplifiedSQLException Defines a wrapper exception that simplifies (clarifies) the message for the SQL exception.
    ValidationException Defines a validation exception that is triggered when a domain object save() fails the validation.
