      • @java.lang.SuppressWarnings("unused")
public class EFrameAssetPipelineService
extends asset.pipeline.micronaut.AssetPipelineService

        Development mode asset pipeline service. Works in dev mode for local (file system) assets, but allows production mode assets from the .jar files (via the asset pipeline manifest). This means no need to restart the server when an asset changes.
