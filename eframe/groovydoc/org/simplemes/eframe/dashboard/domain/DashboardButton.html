Package: org.simplemes.eframe.dashboard.domain

[Groovy] Class DashboardButton

    • org.simplemes.eframe.dashboard.domain.DashboardButton

    • This domain class defines a single configurable button on a dashboard. This allows the user to configure buttons to their needs. These buttons are displayed one after another in the appropriate area of the page. The buttons have configurable labels, styling, title (tooltip) and display sequence.
