Package org.simplemes.eframe.preference

Package org.simplemes.eframe.preference Description

Classes that manage user preference, primarily for the GUI.
  • Interface Summary 
    Interface Description
    PreferenceSettingInterface Defines a user preference setting to contain the user's setting between sessions.
  • Class Summary 
    Class Description
    BasePreferenceSetting Defines the base class for all core preference settings.
    ColumnPreference This holds the specific preferences for a single column in a list.
    ConfigButtonPreference This holds the specific preferences for the initial state of a config toggle button (only one globally).
    DialogPreference This holds the specific preferences for a single dialog from.
    Preference A generic holder for a single user Preference.
    PreferenceDeserializer Serializes a Preference object to JSON for storage.
    PreferenceHolder Defines a user preference object for given user, page and HTML element.
    PreferenceSerializer Serializes a Preference object to JSON for storage.
    SimplePreferenceFactory Factory to build the correct SimplePreference, based on the value type.
    SimpleStringPreference This holds the a simple string preference.
    SplitterPreference This holds the specific preferences for a panel in a splitter for the desired display mode (size).
    TreeStatePreference This holds the list of expanded rows from a tree.
