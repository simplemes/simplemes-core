      • org.simplemes.mes.demand.controller.WorkController 

      • @groovy.util.logging.Slf4j
@io.micronaut.security.annotation.Secured(value: OPERATOR)
@io.micronaut.http.annotation.Controller(value: /work)
class WorkController
extends org.simplemes.eframe.controller.BaseController

        Provides access to the WorkService for starting and completing work on Orders and LSN. Provides API access and client GUI access.

        The WorkService is part of the Stable API.
